# experiments/ — PRISMA Search and Screening Pipeline

## Layout

```
experiments/
├── src/                          Runnable scripts
│   ├── prisma_search.py          Query PubMed (dual-query strategy)
│   └── screening.py              Automated Include/Maybe/Exclude screening
├── data/
│   ├── raw/                      External inputs (immutable)
│   │   └── pubmed_results.csv    227 unique PubMed records
│   └── processed/                Derived, regeneratable
│       └── screening_results.csv 156 eligible after screening
├── results/                      Analysis outputs (regenerated by scripts)
│   ├── domain_counts.md
│   ├── prisma_flow.md
│   ├── top_papers_by_domain.md
│   ├── evidence_synthesis.md
│   ├── screening_summary.md
│   └── search_log.md
└── archive/                      Superseded versions kept for audit trail
    ├── prisma_search_v1.py       Earlier single-query search
    ├── pubmed_results_v1.csv
    ├── domain_counts_v1.md
    ├── prisma_flow_v1.md
    ├── top_papers_by_domain_v1.md
    └── search_comparison.md      v1 vs v2 methodology diff
```

## Reproducing the search

```bash
python src/prisma_search.py   # Writes data/raw/pubmed_results.csv + results/*
python src/screening.py       # Reads data/raw/, writes data/processed/ + results/
```

Both scripts use the Python standard library only. No pip install required.
Python >= 3.10. NCBI Entrez API is rate-limited; no key required.
