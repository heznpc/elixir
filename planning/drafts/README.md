# drafts/ — Superseded Manuscript Forms

Historical Markdown versions of the manuscript kept for reference. **Not the
current source of truth.** The canonical manuscript is `../../paper/main.tex`.

- `outline.md` — Detailed planning outline with section-level evidence mapping.
  Useful for seeing the intended argument structure before LaTeX compression.
- `manuscript.md` — Full prose draft in Markdown. Predates the LaTeX conversion.

Edits here are not reflected in the paper. If a section in these drafts is still
useful, port it to `paper/main.tex` and update here or delete.
